## 数据格式

- **格式**: 纯文本 (`.txt`)，部分 Word 文档 (`.docx`)
- **内容**: 带时间戳和说话人标签的完整文字稿
- **文件数**: 284 个文本文件 + 2 个 Word 文档

示例格式:
```
Ben Horowitz (00:00:00):
领导者做的最糟糕的事情就是在下一个决策上犹豫不决...

Lenny Rachitsky (00:00:23):
担任 CEO、担任创始人是非常困难和痛苦的...
```

## 使用方式

### 用于 AI 分析
这些文字稿非常适合：
- 训练和微调语言模型
- 提取洞察和知识图谱
- 主题聚类和主题分析
- 构建 RAG（检索增强生成）系统

- 向行业领袖学习最佳实践
- 学习产品管理框架
- 了解增长实验和结果
- 获取顶尖人才的职业建议

### 用于研究
- 搜索特定主题或关键词
- 比较不同嘉宾的方法
- 分析趋势变化
- 提取引语和智慧
